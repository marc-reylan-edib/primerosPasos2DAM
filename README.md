PROYECTO README.md

Proyecto de Desarrollo en la Nube de una aplicación de un sistema de colas para parques de atracciones con sistema FIFO.

** APP DE SISTEMA DE COLAS PARA PARQUE DE ATRACCIONES.¡

** En nuestra app funcionará de esta manera:

** 1- El usuario compra la entrada a traves de una web de dicha atraccion 
** 2- El usuario entra a la cola de dicha atraccion 
** 3- Inicia sesión a través de la app usando su entrada 
** 4 - La aplicación le mostrara su turno de cola, le indicará cuanto tiempo de espera debe esperar 
** 5- Cuando le toque su turno le llmaran y podra entrar a la atraccion 
** 6- El usuario sale de la atraccion y podra repetir o subirse a otra atraccion

** Y asi funciona nuestra aplicación de first in first out
